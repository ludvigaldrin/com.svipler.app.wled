WLED Controller for Homey

Control your WLED-powered LED strips directly from Homey. This app enables you to:

- Turn your WLED devices on/off
- Adjust brightness and colors
- Apply effects and color palettes
- Create dynamic playlists
- Control individual LED segments

Works with all WLED-compatible devices (ESP8266/ESP32 running WLED).

Supports both automatic network discovery and manual IP address configuration.
